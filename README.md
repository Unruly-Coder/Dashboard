## Requirements

- [node.js]

## Installation
Copy or clone repository, install dependencies, build project and run server

### Download dependencies

```bash
$ [sudo] npm install
```

### Build project (compile js, css, copy views)

```bash
$ npm run build
```

### Run a server

```bash
$ npm start
```

### Watch .js and .sass files (development mode)

```bash
$ npm run development
```
